#### Custom Adapter SDK란

Custom Adapter SDK의 경우 MobwithSDK 를 직접 사용하는 것이 아니라,  
광고를 이미 사용중인 앱에서 별도로 광고 송출을 위한 추가 기능의 개발 없이,  
기존 사용중인 광고 SDK에 대응하는 Custom Adapter SDK 및 Mobwith SDK를 추가하여 해당 광고 지면에  
Mobwith 광고가 송출 될 수 있도록 하는 기능을 제공하는 SDK 입니다.
