## Supported Environment

---

Deployment Target: iOS 15.6 or higher  
Latest version of Xcode (Xcode 26.0 / Swift 5.3)

The MobWithAD SDK was developed in Swift.  To use the MobWithAD SDK in a Swift-based project, please be sure to use the latest version of Xcode.

- From version 1.3.0, the part that uses external SDKs for mediation functionality has been separated into a dedicated Adapter SDK, so please refer to that section.
- For versions 1.3.0 and above, some important bugs have been fixed, so please set the <b>minimum version to 1.3.2 or higher.</b>
  

## Placement Number(Unit ID) / App Key, etc.

---

For values such as the AppKey required to use external mediation SDKs, as well as placement numbers, please use the values provided to you separately.  
Note that some AppKeys and placement numbers may be affected by the app's Bundle ID, etc., so please take care.
