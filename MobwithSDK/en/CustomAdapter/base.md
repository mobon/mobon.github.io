#### What is the Custom Adapter SDK

The Custom Adapter SDK is an SDK that, rather than using the Mobwith SDK directly,  
provides a feature that lets Mobwith ads be served in a given ad placement by adding the Custom Adapter SDK corresponding to the ad SDK already in use,  
along with the Mobwith SDK, in an app that already uses ads—without developing additional functionality for ad serving.
